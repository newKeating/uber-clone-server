# nuber-server

Server for the Uber Clone
GraphQL, Typescript, NodeJS

## Features

- [ ] Sign In / Sing Up with Facebook
- [ ] Sing In with Email
- [ ] Start Phone Number Verification
- [ ] Complete Phone Number Verification
- [ ] Sing Up with Email
- [ ] Get my Profile
- [ ] Update my Profile
- [ ] Toggle Driving Mode
- [ ] Report Location / Orientation
- [ ] Add Place
- [ ] Edit Place
- [ ] Delete Place
- [ ] See Nearby Drivers
- [ ] Subscribe to Nearby Drivers
